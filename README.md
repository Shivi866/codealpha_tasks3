# Random Quote Generator

Ye ek simple aur interactive **Random Quote Generator** app hai, jise Flutter aur Dart ka use karke banaya gaya hai. Ye app ek button click par screen par naye aur inspiring quotes generate karti hai.

## Features
* **Random Quote Generation:** Har click par naya quote.
* **Modern UI:** Clean aur minimalist design.
* **Responsive:** Mobile devices ke liye fully optimized.

## Technologies Used
* Flutter
* Dart
* Google Fonts (for typography)

## How to Run
1. Make sure you have **Flutter** installed on your system.
2. Clone this repository:
   ```bash
   git clone https://github.com/Shivi866/codealpha_tasks3.git
   ```
